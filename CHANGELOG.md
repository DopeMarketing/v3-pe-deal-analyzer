## [0.1.0] - 2026-03-28

### Added

#### Project Infrastructure
- Initial project scaffold generated by Project Generator
- Next.js 15 application with App Router architecture
- TypeScript configuration with strict mode enabled
- Tailwind CSS styling system with custom design tokens
- Supabase integration for database, authentication, and storage
- Vercel deployment configuration
- Environment variable template (.env.example)
- ESLint and Prettier configuration
- Package.json with all required dependencies

#### Authentication & Security
- Supabase Auth setup with email/password authentication
- Role-based access control (Analyst, Partner, Admin)
- Row Level Security (RLS) policies for all tables
- Protected route middleware
- Authentication layout and page stubs

#### Database Schema
- Complete data model with 14 tables:
  - `users` - User profiles and role management
  - `funds` - Private equity fund information
  - `portfolio_companies` - Portfolio company details and metadata
  - `investments` - Investment transactions and terms
  - `financial_statements` - P&L, balance sheet, cash flow data
  - `financial_metrics` - Calculated KPIs and financial ratios
  - `irr_calculations` - IRR and MOIC calculation results
  - `deal_comparisons` - Side-by-side deal analysis data
  - `alerts` - Performance alerts and notifications
  - `tax_recommendations` - Tax optimization suggestions
  - `executive_dashboards` - Dashboard configuration and layouts
  - `ai_deal_scores` - AI model predictions and confidence scores
  - `integration_configs` - Third-party integration settings
  - `sync_logs` - Data synchronization audit trail
- Database migration files with proper indexing
- TypeScript type definitions for all tables
- Database query utility functions
